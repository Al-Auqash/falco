Catch downloads from your browser
Install the Falco extension once and your browser will hand every download to Falco automatically — a Download File Info window pops up so you can pick where it goes.
  1. Open chrome://extensions (or edge://extensions) in your browser
  2. Turn on Developer mode (toggle in the corner)
  3. Click Load unpacked and select this folder:
The extension button in the browser toolbar toggles catching on/off. Downloads Falco cannot handle (blob:, data:, non-http) stay in the browser. Falco must be running to catch downloads — it starts with Windows by default (Options → General).
